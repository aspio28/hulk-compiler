
# Run

### Execute specific test

```bash
python . --test-id 1 
```

### Execute a range of tests

```bash
python . --test-range 3 5
```

### Execute all tests

```bash
python .
```

### Execute a specific hulk file

```bash
python . --file ./my-code.hulk
```
